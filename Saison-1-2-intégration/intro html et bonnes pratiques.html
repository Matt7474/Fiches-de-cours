HTML & CSS

HTML : Le fond

HTML signifie "HyperText Markup Language" qu'on peut traduire par "langage de balises pour l'hypertexte". Il est utilisé afin de créer et de représenter le contenu d'une page web et sa structure. D'autres technologies sont utilisées avec HTML pour décrire la présentation d'une page (CSS) et/ou ses fonctionnalités interactives (JavaScript).

Les textes

Avec les balises h1-h6 il est possible de définir des titres en respectant une logique de profondeur

  • h1 : 1. : Titre principal
  • h2 : 1.1 : sous-titre
  • h3 : 1.1.1 ...
  • h3 : 1.1.2 ...
  • h3 : 1.1.3 ...
  • h2 : 1.2 : sous-titre
  • h2 : 1.3 : sous-titre

Pour représenter un paragraphe de texte courant la balise p est parfaite

strong et em sont des balises permettant de définir des mots ou expressions signifiants dans un texte courant

CSS : La forme

CSS signifie "Cascading Style Sheets", c'est un langage de feuille de style utilisé pour décrire la présentation d'un document écrit en HTML ou en XML. CSS décrit la façon dont les éléments doivent être affichés à l'écran, sur du papier, en vocalisation, ou sur d'autres supports.

Anatomie d'une règle CSS

Sélecteur

C'est le nom de l'élément HTML situé au début de l'ensemble de règles. Il permet de sélectionner les éléments sur lesquels appliquer le style souhaité (en l'occurence, les éléments p). Pour mettre en forme un élément différent, il suffit de changer le sélecteur.

Déclaration

C'est une règle simple comme color: red; qui détermine les propriétés de l'élément que l'on veut mettre en forme.

Propriétés

Les différentes façons dont on peut mettre en forme un élément HTML (dans ce cas, color est une propriété des éléments p). En CSS, vous choisissez les différentes propriétés que vous voulez utiliser dans une règle CSS.

Valeur de la propriété

À droite de la propriété, après les deux points, on a la valeur de la propriété. Celle-ci permet de choisir une mise en forme parmi d'autres pour une propriété donnée (par exemple, il y a d'autres couleurs que red pour la propriété color).

css /* on est dans un fichier CSS, qui ne contient QUE du code CSS ! Ce fichier peut-être partagé par plusieurs fichiers HTML */ /* commentaire en CSS */ /* pour tout le document on veut utiliser des polices sans empâtement. Pour appliquer cette règle à tous les éléments, on peut l'appliquer à l'élément html. */ html { /* certaines propriétés se répercutent sur les éléments enfants. C'est l'héritage des propriétés. */ font-family: sans-serif; } body { background-color: burlywood; } /* on doit cibler un élément pour lui appliquer des règles (= propriété + valeur). C'est ce que l'on appelle un sélecteur : on sélectionne un ou plusieurs éléments de la page web. */ h1 { /* toutes les règles pour le sélecteur sont entre accolades */ /* une règle de mise en forme est composée d'une propriété, ici on modifie la couleur du texte, et d'une valeur, on met la couleur du texte en rouge. On termine une règle par un point-virgule */ color: red; background-color: beige; /* taille de police à 40 pixels (unité : px) le pixel est une unité fixe qui est la même sur tous les écrans. C'est une unité absolue. */ font-size: 40px; /* on peut choisir l'alignement du texte */ text-align: center; } h2 { color: green; } /* on veut une police avec empattement pour les titres 3 */ h3 { font-family: serif; } /* on peut modifier la forme du strong */ strong { font-weight: normal; /* couleur de fond, yellow est la couleur jaune */ background-color: yellow; } em { font-style: normal; text-decoration: underline; } footer { font-size: 12px; text-align: center; } /* on peut ne sélectionner que les éléments d'un élément. Ici on sélectionne uniquement les liens (a) qui sont dans le pied de page (footer). On utilise la syntaxe élément-parent élément-enfant */ footer a { color: cyan; }
